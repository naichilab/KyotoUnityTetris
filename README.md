京ゆにテトリス
---

###概要
このプロジェクトは*「京都Unityもくもく会 #14 」*にて作られたものです。

テトリスをUnityで作ることで、Unityを使ったゲームの作り方の基礎を固めることを目標としています。

そのため、このプロジェクトはゲームとしての面白さを追求したものではなく、技術訓練用のプロジェクトとなっています。

Unityのバージョンは、**Unity5.3.4f1**を使って作成されています。
